About CV analysis and Ellegibility Prediction system

Our system presents a web-based system for analyzing job applicants' CVs and predicting their eligibility for a given job, using data mining concepts. The system employs a combination of text mining and machine learning techniques to extract relevant information from CVs, such as education, work experience, skills, and achievements.

The extracted information is then used to build a predictive model that can assess an applicant's eligibility for a specific job based on predefined criteria. The proposed system aims to automate and simplify the recruitment process by reducing the time and effort required to manually screen and evaluate job applications. The system has been evaluated using a dataset of real-world CVs and job descriptions, and the results demonstrate its effectiveness in accurately predicting the eligibility of job applicants. Overall, this paper provides a practical and efficient solution for improving the recruitment process in organizations.

We are initiated to develop this project because the Arba Minch University Human Resource department is having difficult condition in analyzing CV of each candidate applying for the particular job that takes long time and more human power. The main purpose of this project is to develop and implement a computerized CV analysis system to help HR to provide reliable services.

These systems tend to us to find possible solution about the problem, it helps applicants to access the system easily and also provide Eligibility prediction model developed by using machine learning.
